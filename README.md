# Ischemic Heart Lab — Interactive Website

Open `index.html` directly in a modern browser. No server, package installation, or internet connection is required for the core website.

## Included

- 18 interconnected learning modules
- Responsive sidebar and mobile navigation
- Search, bookmarks, local progress, print view, and light/dark theme
- Browser text-to-speech with preference for an English (UK) voice when available
- Interactive supply–demand, symptom, testing, medication, revascularization, ACS, STEMI, ECG, complication, and discharge tools
- Six integrated clinical cases
- 36 true flip flashcards with mouse and keyboard controls
- 20-question exam quiz with explanations and saved progress
- Local copy of the source PDF and links to authoritative ESC and ACC/AHA guideline hubs

## Notes

Text-to-speech voice availability depends on the operating system and browser. Chrome or Edge usually provides the best voice selection.

This is an educational study aid and not a patient-specific diagnostic or treatment protocol.
